# Zhongde Boiler Bangladesh 🏭

Official Bangladesh Representative of **HeNan ZhongDe Boiler Equipment Co., Ltd.** — China's premier industrial boiler manufacturer.

## 🌐 Live Site
Deployed at: [https://zhongde-bangladesh.vercel.app](https://zhongde-bangladesh.vercel.app)

## 📋 About This Site
A professional business website for Zhongde Boiler's Bangladesh operations, targeting:
- Bangladeshi factory buyers (garments, textiles, food, pharma)
- Zhongde Boiler's international sales team
- B2B industrial procurement managers

## 🛠️ Tech Stack
- Pure HTML5 + Vanilla CSS + JavaScript (zero dependencies)
- Vercel Static Hosting
- Fully responsive (mobile-first)
- SEO optimized
