## 📋 About This Project
A professional, credible business website for Zhongde Boiler's official Bangladesh operations — designed to build trust with both Bangladeshi factory buyers and Zhongde's international sales team in China.

**Target audiences:**
- Bangladeshi RMG factory managers, procurement officers
- Textile, food, pharmaceutical, construction industry buyers
- ZhongDe Boiler's international business development team

## 🚀 Deploy to Vercel

### Option 1: Vercel Dashboard (Recommended)
1. Go to [vercel.com/import](https://vercel.com/import)
2. Import this GitHub repo: `munim430-ai/zhongde-bangladesh`
3. Click Deploy — it auto-detects the vercel.json config

### Option 2: CLI
```bash
npm i -g vercel
vercel
```
